Discovering the different display settings with block, inline and inline-block.

This lesson you will learn about how items on your websites are displayed using the display setting. We will cover how the display settings work and the most common display settings - block, inline and inline-block.

What is display?

Display is a term used in css to determine how they content will be displayed, as there are many ways to display an item. For example if we have an item and we don’t want it to be visible we can use the css code: Display: None; lets say for some reason we want to display the item inside a table like grid, we could use the code: Display: Table
By default all items use display: block; This means that each block element will start on a new line and take up all the available width inside its parent element. For example with block elements you can choose the width, but by default it will push the next element down.

So there are many different options on how to display an item using the code display. This blog explores two of the most commonly used display options:
Display: Inline and Display: Inline-block;.

What is Display: Inline ?

Inline is just that. It’s on the same line.
Its the default setting for a text span or a text link. Inline content ignores any values you’ve entered for dimensions eg. Making the width 300px will be ignored if displayed inline. Instead, It can only be changed using text settings like font size and font weight. This video is the perfect example showing elements on the same line being adjusted by text settings. Display:inline is great for hyper links or text styling within a paragraph.

What is Display: Inline - block ?

Inline block is as wide as the content inside, inline elements flow on the same line and wrap to the next line when it hits the boundary of the parent item. With inline block elements you can adjust the size of the item by changing the width and so on. Aswell as this Inline elements react to the text alignment of the parent elements. This video shows elements being added to the same line, until they reach the boundary and move to the line below. Display: inline - block is good for wrapping multiple images in a container.
